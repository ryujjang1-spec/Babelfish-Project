# Babelfish_온디멘드 컨시어지

Babelfish는 고객의 요청을 이해하고, 제휴 서비스와 연결해 실행까지 돕는 AI 온디멘드 컨시어지 데모입니다.

## 주요 기능

- 아이나비 M 택시 호출
- Babelfish 제휴 병원 예약
- 자동차 정비 및 검사 예약
- 아이나비 블랙박스 장착 연결
- 칼트윈 틴팅 필름 시공 연결
- 상품 구매 및 가격 비교 지원
- 음성 기반 실시간 상담 데모

## 실행 방법

npm install

npm run dev

## 로컬 접속

http://localhost:3000

http://localhost:8787/health

## 환경변수

.env 파일에 아래 값을 설정합니다.

OPENAI_API_KEY=your_openai_api_key
PORT=8787
NEXT_PUBLIC_API_BASE_URL=http://localhost:8787

## 주의사항

.env 파일은 GitHub에 올리지 않습니다.
